
To install fontmatrix in Ubuntu 16.04 

First download libicu52 from (chose 32 or 64 bits):

http://packages.ubuntu.com/trusty/libicu52

install it with gdebi

Next download fontmatrix from (chose 32 or 64 bits):

http://packages.ubuntu.com/trusty/fontmatrix

install it with gdebi

OK
